# Slot Booking System

A Django-based web application that allows users to book available slots.

## Features
- User login
- View available slots
- Book slot
- Cancel booking
- Prevent double booking

## Tech Stack
- Python
- Django
- SQLite
- HTML/CSS

## Setup

git clone https://github.com/yourusername/slotbooking.git
cd slotbooking

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
